InSAR · Sentinel-1 SARvey / ISCE2 / MiaplPy PINN (Euler-Bernoulli + FEM) FRAM · CRI GPLv3
GitHub 저장소 → 빠른 시작
⚠️ Status — 연구용 프로토타입 (Research prototype). 실 Sentinel-1 → SARvey → PINN → FRAM 전 파이프라인 실행 + 해석해 기반 PINN/FEM 검증(오차 <0.1%)은 실증됨. 다만 현장 실측(AVT·GNSS/수준측량)·상용 FEM·실 붕괴/점검 라벨 검증은 미수행이고, 붕괴확률은 합성 Morandi 라벨 기반이다. 표시되는 변위·CRI·붕괴확률은 파이프라인 산출이며 실무 교량 안전 판정용이 아니다.

미리보기

정자교 실 Sentinel-1 데이터를 다운로드→ISCE2 코레지→MiaplPy→SARvey→PINN→FRAM 으로 처리한 실제 산출.

라이브 대시보드

Streamlit — 교량 선택·SLC 다운로드·처리·위험 진단·리포트를 한 화면에.

무엇을 하나

① InSAR

OSM 교량 선택 → ASF SLC 검색·트랙 선별 → SARvey 시계열 → 변위 추출. 기준점·온도회귀·대기보정으로 정확도 보정.

② PINN

Euler-Bernoulli PDE + FEM 모달로 변위를 열팽창/하중/침하/이상으로 분해, 절대 EI·α 역산. 해석해 검증(오차 <0.1%).

③ FRAM

기능공명분석으로 공명위험지수 CRI + 4단계 경보. isotonic 캘리브로 붕괴확률 매핑, N-K 함수망.

대시보드

Streamlit — 교량 선택·SLC 다운로드·처리·위험 히트맵·시계열·리포트(PDF)·다중 교량 포트폴리오.

파이프라인

OSM·ASF·ERA5 선별 → (WSL2) ISCE2 코레지 → MiaplPy → SARvey → Track H5
→ InSAR 정합 → PINN 구조해석 → FRAM 공명위험 CRI → 대시보드

빠른 시작

pip install -e ".[dev]"
python -m inframon --doctor          # 환경 진단
python -m inframon --demo            # 합성 데모 → data/project.h5 + CRI
python -m streamlit run src/inframon/dashboard/app.py   # 대시보드

실 InSAR 처리(ISCE2/MiaplPy/SARvey)는 WSL2/Linux 필요. 코어·대시보드는 합성으로 동작.
